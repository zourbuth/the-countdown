On Expiry

On Expiry or onExpiry is a callback function that is invoked when the countdown reaches zero.

This is an example using On Expiry to run a JavaScript:

<script type="text/javascript">
    function myExpiry() {
        alert( 'Hey, I am expired' ); 
    }
</script>

The code above can be placed in the widget Frontpage Footer field as screenshot below.

On Tick

A callback function name that is invoked each time the countdown updates itself. The array of current countdown periods (int[7] - based on the format setting) is passed as a parameter: [0] is years, [1] is months, [2] is weeks, [3] is days, [4] is hours, [5] is minutes, and [6] is seconds.

<script type="text/javascript">
    function myTick(periods) {
        console.log( periods );
    }
</script>
