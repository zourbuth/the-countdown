A plugin for displaying counter for your site and do action after or before it hit the expiration.

== Description ==

The countdown functionality can easily be added to your site. With counting down and up functionality, gives you a full control to your counter. 
This plugin comes with  multi-instances sidebar widget, packed with bunch of options. All styles and scripts area pulled in the header section. 
Every each counter have their own selector that enables you to create spesifics style or script to the selector.

<h3><strong>Key Features & Options</strong></h3>
<ul>
<li>Date Picker for your until or since counter.</li>
<li>Expiry Url a URL to load upon expiry, replacing the current page.</li>
<li>Expiry Text text to display upon expiry, replacing the countdown.</li>
<li>Date Format easy to use custom date format for display.</li>
<li>Custom Labels the display texts for the counters, singular, plural name, or with compact.</li>
<li>Tick Interval interval in seconds between on tick callbacks</li>
<li>Template styles custom background and colors.</li>
<li>Intro & Outro text, if you want to add additional text or HTML.</li>
<li>Custom Style & Script if you want to add additional CSS style or javascript per widget ID, custom post, or using global value.</li>
</ul>

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin manually.
1. Go to the menu 'Plugins' -> 'Install' and search for 'The Countdown'
2. Click 'install'.
